Overview of ILEX

Welcome to ILEX, a small useful toolkit for analysing ASKAP Fast Radio Bursts (FRBs). This toolkit has a number of features, including

  • Polarisation analysis

  • High Time resolution analysis

  • Quality plotting

To start learning how to use ILEX, move to the “Tutorials” section of the documentation.
